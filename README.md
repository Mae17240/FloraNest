# FloraNest Plant Mobile application. 
FloraNest is a Plant Scanning and deterction IOS application. 
The project features a CoreML model embedded into the application to differenceiate various outdoor and indoor species. 

# FloraNest Features:
- Accurate Plant Identification.

## CoreML Model.
- Identification of 38 indoor and outdoor species.
- Embedded and Trained CoreML Model that can be ran offline.

## Account System
- Account system: Storage of plants and user data.
- Keychained user data.

## Map API
- View plants whithin your local area.
